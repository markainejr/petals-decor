## Setup Instructions
```bash
# Clone the repository
git clone https://github.com/markainejr/petals-decor.git

# Navigate to the project directory
cd petals-decor

# Install dependencies
npm install

# Start the development server
npm run dev
```
Visit the application at `http://localhost:5173`
```
